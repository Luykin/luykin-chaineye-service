基本信息

Fred Ehrsam 是 Paradigm 的联合创始人。此前，Fred 共同创立了美国最大的加密货币公司 Coinbase，并于 2012 年至 2017 年担任总裁一职。Fred 于 2011 年购买了他的第一个比特币，并一直是该领域一些最具成长性公司的天使投资人。在加入 Coinbase 之前，Fred 是纽约高盛的外汇交易员。Fred 以优异的成绩从杜克大学获得计算机科学和经济学学士学位。

Radicle 是一个基于开放协议的去中心化代码协作网络。它使开发人员能够在不依赖受信任的中介的情况下就代码进行协作。 Radicle 旨在为中心化代码协作平台（或“锻造”）提供类似的功能，同时保留 Github 的点对点性质。 Radicle 不使用用户帐户和登录名，而是使用公钥密码术来识别项目及其合作者。 Radicle 不是由公司托管和拥有的问题跟踪器，而是使用种子节点。Radworks 是 Radicle 背后的社区管理网络。
